# GMember安装与使用

# 环境支持

不支持IE11及之前版本。

# 当前最新版本

GMember目前还处于快速开发迭代中：npm: v0.1.5

# 通过NPM安装

我们推荐使用包管理器的方式安装，它能更好地和 vite, webpack 打包工具配合使用。

npm install gmembertest --save
1

如果你的网络环境不佳，建议不要使用。

# 使用组件

# 完整引入所有组件

main.ts

import { createApp } from 'vue'
import App from './App.vue'
import Gmember from 'gmembertest'
import 'gmembertest/lib/gmembertest.css'

createApp(App).use(Gmember).mount('#app')
