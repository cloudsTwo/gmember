# Color 色彩

为了统一多个平台多个项目产品，制定了一套产品级色彩体系。

# 主色

采用严肃、理性、专业的科技蓝作为产品主色。

Brand
#2878FF
Hover
#6199F8
Click
#1C67E4

# 功能色

使用于页面链接、成功、警告、错误、信息提示等具体场景中，功能色的选取要遵循用户对色彩的基本认知，同系列产品功能色尽量保持一致，以免干扰用户的认知体系。

Success
#65CA32
Warning
#EEB107
Error
#F76D4C
Info
#909399

# 中性色

主要用于文字、背景底色、边框、分割线、图标等，运用不同的中性色来表现层次结构。

标题
#303133
正文
#606266
辅助信息
#909399
图标
#B0B3B8
失效文字
#C0C4CA
边框
#DCDEE3
分割线
#E8EAED
背景
#F5F5F5
